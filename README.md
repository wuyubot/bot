# wuyubot

#### 介绍
一个普通的nonebot实现

#### 软件架构
基于nonebot2 魔改而来


#### 安装教程

1.  装Python3.10+，然后装redis，poetry
2.  克隆到本地
3.  poetry install装依赖

#### 使用说明

0.  运行方法：poetry run python bot.py(linux/macos 考虑用python3替换python)
1.  运行一次，然后会生成默认配置，填完后再次运行
2.  xxxx
3.  xxxx
